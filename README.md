## Getting Started

To run the project in your local environment:

```bash
# Clone the repository
git clone https://github.com/yourusername/e-commerce-website.git
cd e-commerce-website

# Install dependencies
npm install
# or
yarn install

# Start the development server
npm run dev
# or
yarn dev
```
